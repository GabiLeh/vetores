# Vetores
##    Vetores são unidimensionais e matrizes são multidimensionais
Em C são conjuntos contíguos de dados do mesmo tipo, acessados por índices que começam em 0. São declarados em ``nome[tamanho]`` e manipulados com laços ```for``` para percorrer linhas e colunas. Matrizes são "vetores de vetores", armazenados de forma linear na memória
